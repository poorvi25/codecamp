About HTML

HTML is the standard markup language for creating Web pages.


  • HTML stands for Hyper Text Markup Language
  • HTML describes the structure of Web pages using markup
  • HTML elements are represented by tags
Basic Tags
  • All HTML documents must start with a document type declaration:
    <!DOCTYPE html>
  • The HTML document itself begins with
    <html> and ends with </html>
  • The visible part of the HTML document is between
    <body> and </body>
Semantic Tags

A semantic element clearly describes its meaning to both the browser and the developer.

Examples of non-semantic elements: <div> and <span> - Tells nothing about its content.

Examples of semantic elements: <form>,<table>,and <article> - Clearly defines its content.

HTML Elements

An HTML element usually consists of a start tag and end tag, with the content inserted in between:

<tagname>Content goes here...</tagname>

Some of them are: <p></p>,<h1></h1>,<body><body>,<div><div>etc.....

HTML Attributes
  • All HTML elements can have attributes
  • Attributes provide additional information about an element
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

    • Some of Examples are:

      Href Attributes:
    • HTML links are defined with the <a> anchor tag. The link address is specified in the href attribute:

      • <a>href="https://website.html"></a>

      Src Attribute:

    • HTML images are defined with the <img> image tag
    • The filename of the image source is specified in the src attribute:

      • <img> src="xyzzz.jpg">

      Like this are many More

HTML Headings
  • Headings are defined with the <h1> to <h6>tags.
  • <h1>defines the most important heading.<h6>defines the least important heading.
HTML Style
  • Setting the style of an HTML element, can be done with the style attribute.
  • The HTML style attribute has the following syntax:

    • <tagname style="property:value;">

  • The property is a CSS property. The value is a CSS value.
